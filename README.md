# 🌦️ Weather Prediction

## 📌 Project Overview
This project is a Machine Learning-based Weather Prediction system developed using Python in Google Colab. The model analyzes weather-related data and predicts weather conditions based on the input features.

## 🚀 Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Machine Learning model training
- Weather prediction based on input data
- Model evaluation using performance metrics

## 🛠️ Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 📂 Project Structure
```
Weather_Prediction/
│── AIML_CT_PROJECT.ipynb
│── README.md
│── dataset.csv (if included)
│── requirements.txt (optional)
```

## ▶️ How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/Kalai-16/Weather_Prediction.git
   ```
2. Open the notebook in Google Colab or Jupyter Notebook.
3. Install the required libraries if needed:
   ```bash
   pip install pandas numpy matplotlib seaborn scikit-learn
   ```
4. Run all the notebook cells in sequence.

## 📊 Output
The model predicts weather conditions based on the provided input data and displays the prediction results along with model evaluation.
